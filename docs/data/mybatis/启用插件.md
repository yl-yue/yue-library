## 启用插件 <!-- {docsify-ignore} -->
```java
@Bean
public MybatisPlusInterceptor mybatisPlusInterceptor(List<InnerInterceptor> innerInterceptorList) {
    MybatisPlusInterceptor plusInterceptor = new MybatisPlusInterceptor();
    // 保留内置插件启用
    plusInterceptor.setInterceptors(innerInterceptorList);
    // 添加乐观锁插件
    plusInterceptor.addInnerInterceptor(new OptimisticLockerInnerInterceptor());
    // 添加非法SQL拦截器
    plusInterceptor.addInnerInterceptor(new IllegalSQLInnerInterceptor());
    // 防全表更新与删除插件
    plusInterceptor.addInnerInterceptor(new BlockAttackInnerInterceptor());
    // 数据权限插件
    ...
    // 动态表名插件
    ...
    // 数据变动记录插件
    ...
    return plusInterceptor;
}
```

启用新插件时，需要**保留内置插件启用**，否则多租户、逻辑删除等特性会失效。
