# 数据脱敏

　　日志模块内置数据脱敏能力，防止敏感信息泄露到日志数据库。

## 1. 默认脱敏策略

默认脱敏策略对以下密码类字段置空处理：

| 字段名 | 脱敏方式 |
|--------|----------|
| password | 置空 |
| pwd | 置空 |
| passwd | 置空 |
| secret | 置空 |
| accesstoken | 置空 |
| refreshtoken | 置空 |

> 脱敏仅针对 `request_param` 字段，`response_result` 不做脱敏（通常不含敏感信息）。

## 2. 自定义脱敏策略

实现 `LogMaskStrategy` SPI 接口，注册为 Spring Bean：

```java
@Component
public class CustomLogMaskStrategy implements LogMaskStrategy {

    @Override
    public String mask(String param) {
        if (StrUtil.isBlank(param)) {
            return param;
        }

        try {
            JSONObject json = JSONObject.parseObject(param);
            // 手机号中间4位掩码
            if (json.containsKey("phone")) {
                String phone = json.getString("phone");
                json.put("phone", phone.substring(0, 3) + "****" + phone.substring(7));
            }
            // 身份证号中间8位掩码
            if (json.containsKey("idCard")) {
                String idCard = json.getString("idCard");
                json.put("idCard", idCard.substring(0, 4) + "********" + idCard.substring(12));
            }
            // 邮箱中间部分掩码
            if (json.containsKey("email")) {
                String email = json.getString("email");
                int atIndex = email.indexOf("@");
                json.put("email", email.substring(0, 2) + "***" + email.substring(atIndex));
            }
            return json.toJSONString();
        } catch (Exception e) {
            return param;
        }
    }
}
```

> 注册自定义 `LogMaskStrategy` Bean 后，默认脱敏策略将被替代。

## 3. 配置项

```yml
yue:
  data:
    log:
      mask-enabled: true  # 是否启用脱敏（默认 true）
```

关闭脱敏后，请求参数将原样存储到数据库。
