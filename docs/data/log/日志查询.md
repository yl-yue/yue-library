# 日志查询

　　日志模块提供面向业务管理后台的查询接口，支持分页查询与详情查看。

## 1. 启用查询接口

查询接口默认不启用，业务项目需显式开启：

```yml
yue:
  data:
    log:
      query-api:
        enabled: true
```

> 查询接口仅在直连写库模式（`DefaultLogStorageProvider`）下可用。HTTP 转发模式下，查询应直接访问日志服务。

## 2. 登录日志查询

### 分页查询

```
GET /auth/v1/log/loginLog/page
```

**请求参数：**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| pageNum | Integer | 是 | 当前页 |
| pageSize | Integer | 是 | 每页条数 |
| username | String | 否 | 登录账号（模糊匹配） |
| status | Integer | 否 | 登录状态（0=失败, 1=成功） |
| startTime | Long | 否 | 开始时间 |
| endTime | Long | 否 | 结束时间 |

**响应示例：**
```json
{
  "flag": true,
  "code": 200,
  "msg": "操作成功",
  "data": {
    "total": 100,
    "list": [
      {
        "id": 1,
        "username": "admin",
        "ip": "192.168.1.100",
        "loginTime": 1778757065542,
        "status": 1,
        "msg": "登录成功",
        "requestParam": "{\"username\":\"admin\"}",
        "createUser": "管理员",
        "createUserId": 1,
        "createTime": 1778757065542,
        "tenantSysId": "sys",
        "tenantCoId": "co"
      }
    ],
    "pageNum": 1,
    "pageSize": 10,
    "pages": 10
  }
}
```

### 详情查询

```
GET /auth/v1/log/loginLog/getById?id={id}
```

**响应示例：**
```json
{
  "flag": true,
  "code": 200,
  "msg": "操作成功",
  "data": {

    "username": "admin",

    "loginTime": 1778757065542,
    "status": 1,
    "msg": "登录成功",
    "requestParam": "{\"username\":\"admin\"}"
  }
}
```

## 3. 操作日志查询

### 分页查询

```
GET /auth/v1/log/operLog/page
```

**请求参数：**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| pageNum | Integer | 是 | 当前页 |
| pageSize | Integer | 是 | 每页条数 |
| title | String | 否 | 操作标题（模糊匹配） |
| bizType | String | 否 | 业务分类（精确匹配） |
| operType | String | 否 | 操作类型：C/R/U/D（精确匹配） |
| username | String | 否 | 操作账号（模糊匹配） |
| status | Integer | 否 | 操作状态（0=失败, 1=成功） |
| startTime | Long | 否 | 开始时间 |
| endTime | Long | 否 | 结束时间 |


## 4. 安全前缀说明

查询接口使用 `auth` 安全前缀（`/auth/v1/log/`），表示需认证的管理后台接口。与存储接口的 `lan` 前缀（`/lan/v1/log/`）职责区分：

| 前缀 | 用途 | 说明 |
|------|------|------|
| `auth` | 查询接口 | 需认证，面向业务管理后台 |
| `lan` | 存储接口 | 局域网内部，面向微服务日志转发 |

## 5. Service 层直接调用

如不使用 Controller 接口，也可直接在 Service 层调用查询方法：

```java
@Resource
private LoginLogService loginLogService;

@Resource
private OperLogService operLogService;

// 登录日志分页查询
LoginLogPageIPO ipo = LoginLogPageIPO.builder()
        .username("admin")
        .status(1)
        .startTime(startTime)
        .endTime(endTime)
        .build();
PageInfo<LoginLogEntity> page = loginLogService.pageLoginLog(ipo);

// 操作日志分页查询
OperLogPageIPO operIpo = OperLogPageIPO.builder()
        .title("用户")
        .operType("C")
        .build();
PageInfo<OperLogEntity> operPage = operLogService.pageOperLog(operIpo);

// 详情查询
LoginLogEntity loginLog = loginLogService.getLoginLog(id);
OperLogEntity operLog = operLogService.getOperLog(id);
```
